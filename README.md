# Ederra Piscine - Site vitrine

Site vitrine en HTML statique. Aucun build requis.

## Structure

```
cursor/
├── index.html        # Accueil
├── prestations.html  # Nos Prestations
├── contact.html     # Contact + modales
├── css/style.css    # Styles personnalisés
├── js/site.js       # Menu mobile, carousel, modales
├── images/          # Photos du site
├── logo/            # Logo Ederra
└── favicon.svg
```

## Lancer le site

```bash
npm run dev
```

Ouvre http://localhost:3000

Alternative : ouvrez `index.html` directement dans le navigateur, ou utilisez tout serveur HTTP statique (Live Server, Python `http.server`, etc.).

## Technologies

- HTML5
- Tailwind CSS (CDN)
- JavaScript vanilla
